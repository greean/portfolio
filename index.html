Projects

Block Breaker

A version of the classic 'Block Breaker/ Arkaniod' game, created using javascript for HTML5 Canvas.

After teaching myself a few fundamentals of HTML5 Canvas, I then developed this game where you have to break blocks in the top half of the screen by bouncing a ball off the side of the screen and a 'paddle' near the bottom of the screen, controlled by the user.

I've so far included a score, lives for the game and a pause feature - press the 'p' button. However, the game will be developed further to include bonuses that drop from the broken blocks and speed up/ slow down the game.

 

CyberPet: Gizmo

'CyberPet: Gizmo' is an app where you have to make decisions on how to look after your pet 'Gizmo'.

Built using javascript to manipulate the DOM, you are given 3 random options to choose from in order to increase Gizmo's health or happiness. However, some of the options will have the opposite effect.

 

Slack Clone

A 3-day team project to create an app using React, based on the communication platform Slack.

As well as replicating the look of the Slack platform, due to the time restraints of the project we decided to concentrate on the following main features:

  • Ability to add a message to the thread

Operation Raven

After a 2 day crash course in Python, this team project required us to create a site incorporating a django database.

Using spreadsheet data regarding UFO sighting events in the UK, the brief was to build a database and display the following:

  • A page listing all events
  • A page that shows an individual event
  • A page that lists all locations
  • A page that lists all events for a given location

 

Dice Game

The brief for the project was to create a 1 player dice game using javascript and DOM.

  • Roll the dice to begin the game
  • The dice appears on the first roll
  • When a dice is rolled, the score adds up
  • Once the score reaches 20, you win.
  • However, if you roll a ‘1’ then you lose
  • If the player won or lost, the player can start the game again

A further challenge was to then create a 2 player version of the game.
